# Module 1: The Robotic Nervous System (ROS 2)

Welcome to Module 1 of the Physical AI & Humanoid Robotics textbook. This module covers the fundamentals of Robot Operating System 2 (ROS 2), which serves as the nervous system for robotic applications.

## Overview

This module is divided into two main sections:

1. **Weeks 1-2**: Introduction to Physical AI and basic sensors/perception
2. **Weeks 3-5**: ROS 2 Fundamentals including architecture, packages, nodes, topics, and services

## Learning Objectives

By the end of this module, you will:
- Understand the fundamentals of Physical AI and embodied intelligence
- Master ROS 2 architecture and core concepts
- Be able to create and manage ROS 2 packages
- Implement nodes, topics, and services
- Work with URDF robot descriptions
